# Joshua Chua — Portfolio

**Live:** [https://joshpchua.github.io/](https://joshpchua.github.io/)

## Purpose

Recruiter-focused portfolio for IT Systems Operations & Automation roles. Designed to present projects, skills, and experience in a clean, professional format optimised for quick scanning by hiring managers and technical recruiters.

## Tech Stack

- HTML, CSS, JavaScript — no frameworks
- Fonts: Inter, Geist, Geist Mono (Google Fonts)

## Local Development

No build step required. Open `index.html` in any modern browser.

## Deployment

Push to the `main` branch. GitHub Pages auto-deploys from root.

## File Structure


## Accessibility

- Semantic HTML elements
- Skip-to-content link
- Full keyboard navigation
- `prefers-reduced-motion` support
- WCAG-compliant contrast ratios

## Performance

- No build step or bundler
- Minimal JavaScript
- Lazy-loaded images
